# Standard



## Install

npm

```sh
npm install standard eslint-config-standard -D
```

yarn

```sh
yarn add standard eslint-config-standard -D
```



## Configuration

`.eslintrc.json`

```json
{
  "root": true,
  "env": { 
    "es6": true
  },
  "extends": "standard",
  "parserOptions": {
    "sourceType": "module"
  }
}
```
